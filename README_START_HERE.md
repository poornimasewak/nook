## 🚀 Quick Start (3 Steps)

### 1. Start the Server

```bash
npm run dev
```

### 2. Open Browser

Visit: **http://localhost:3000/login**

### 3. Test Login

1. Select country code (e.g., 🇺🇸 +1)
2. Enter phone number (e.g., `1234567890`)
3. Click "Continue"
4. **Check your terminal** for OTP code
5. Enter OTP in browser
6. Click "Verify"

**Done!** ✅
